# Smart Attendance Management System

A web-based Smart Attendance Management System built using HTML, CSS, and JavaScript for managing student attendance, viewing records, and generating reports using Local Storage.

## Features

* Mark student attendance
* View attendance history
* Generate attendance reports
* Store attendance data using Local Storage
* Responsive and user-friendly interface

## Technologies Used

* HTML
* CSS
* JavaScript
* Browser Local Storage
